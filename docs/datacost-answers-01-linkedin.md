**DataCost Answers #1: How do you check and recharge a SIM inside a remote router?**

A South African customer asked how to check the remaining data on a SIM installed in a router at a remote site—and how to top it up without travelling back to the site.

The best first step is to link the router SIM's cellphone number to the network provider's app or web account. That lets you check the network-side balance and buy a bundle remotely.

The router's own USSD page can be a useful backup, but it is not available on every router or firmware version. Set up and test remote access while you can still read the router's SMS inbox, because registration may require an OTP.

We compared the practical routes for MTN, Vodacom, Telkom, Cell C and Rain, and included guidance for teams managing several customer sites.
