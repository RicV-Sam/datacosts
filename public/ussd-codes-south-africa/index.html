Frequently asked questions

Useful context for users who land here from search and want a fast answer.

Which network USSD code should I use to check my balance in South Africa?

It depends on your operator. Common examples on this page include *136# for MTN, *135# for Vodacom, *188# for Telkom and *101# for Cell C.

Can I tap these codes directly from my phone?

Yes. The dial buttons use telephone links so mobile users can open their dialer immediately. You can also copy a code first.

Does Rain use USSD codes like the other networks?

No. Rain is handled mainly through the app and website, which is one reason it should be treated differently from the traditional mobile SIM operators.
